Estrutura HTML

Estrutura de Um Documento

  • head é o elemento que contém as informações gerais do Documento.
  • meta é o elemento utilizado para representar outros metadados ou informações adicionais od documento.
  • title contém o título do Documento.

    • Organização da Página

  • h1-h6 são elementos que representam cabeçalhos, sendo o h1 o mais alto até o h6, mais baixo.
  • p representa parágrafos
  • div é uma espécie de contêiner para organizar o conteúdo na página.
  • span é outro contêiner mas inline.
  • li é usado para representar uma lista organizada.
